🧠 How Scrum Master Estimates Story Points

  • Complexity → How difficult is the task?
  • Effort → How much work is required?
  • Risk → Unknowns / dependencies
  • Priority → High priority = more effort
  • Type → Backend / UI / Database / Security

📌 Example

Task: Login system with database

  • Database integration
  • Authentication logic
  • Error handling

  • Rule Engine adjusts result
  • LLM explains reasoning

🔄 Final Flow

Input → Jira → AI → Rules → LLM → Output
